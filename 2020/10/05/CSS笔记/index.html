  4. 其他属性

一些还没记牢的标签属性

table

cellspacing: cell间的空白

cellpadding: 内容与cell边缘的距离

border-collapse: 合并相邻边框

font

text

text-align:

text-indent: ? px | em(文字单位)

text-decoration:

text-shadow:

文字垂直居中

line-height = height

剩下的大脑都能记住（大概）

选择器

后代选择器

ul li { } （ul中所有的li，含2代以上）

子选择器

ul>li { }（ul第一代中的li）

并集选择器

div, p { }

伪类选择器

a:link(未被访问) a:visited(已被访问) a:hover(鼠标经过) a:active(鼠标按下未抬起) (固定顺序)

input:focus(获得光标的输入栏)

行块元素

转换: display: block | inline | inline-block;

背景

背景色

background-color

透明效果

background: rgba(0,0,0,x(alpha透明度));

背景图片

background-image: url( );

背景重复

background-repeat: repeat | no-repeat | repeat-x | repeat-y;

背景位置

background-position: left top | 0 0;(如果只指定一个方位，另一方默认居中对齐)

背景滚动

background-attachment: scroll | fixed

继承性

后代会继承先代的格式

选择器类型 权重
继承 | * 0,0,0,0
元素选择器 0,0,0,1
(伪)类选择器 0,0,1,0
ID选择器 0,1,0,0
行内样式 style=” “ 1,0,0,0
!important ∞

单盒布局

边框

(border, border-top, border-bottom… )

border-width

border-style: none | solid | dashed | dotted

border-color

可简写，无顺序要求

内边距

padding(-left, -right， …):

外边距

margin:

块级盒子水平居中

  1. 设定盒子宽度
  2. 将盒子左右外边距设置为auto，margin: 1 auto;

行内元素、行内块元素水平居中

​ 上级元素添加text-align = "center"

嵌套块元素塌陷

​ 加大子元素的外边距，会导致父元素外边距也增大

​ 解决方案：

  • 为父元素定义上边框

  • 为父元素定义上内边距

  • 为父元素添加overflow: hidden

清除内外边距

圆角边框

border-radius: length

盒子阴影

滚动锁定

display: fixed

浮动

float: none | left | right

特性

  • 浮动元素会脱离标准流(脱标)，不再保留原来的位置

  • 浮动元素会在一行内显示，装不下换行，并且顶部对齐

  • 浮动元素具有行内块元素的特性

清除浮动

  • 父元素添加overflow: hidden

  • :after伪元素法

  • :after :before双伪元素法

定位

静态定位

position: static

相对定位

position: relative

  • 原地占位，不脱标

绝对定位

position: absolute

  • 若无父元素或父元素无定位，则以整个网页为参照
  • 以最近的带有定位的父元素为标准
  • 不原地占位，脱标

固定定位

position: fixed

  • 以浏览器的可视窗口为参照
  • 不原地占位，脱标

元素的隐藏

display

display: none | block

  • 占位

visibility

visibility: visible | hidden

  • 不占位

overflow

overflow: visible | hidden | auto | scroll

  • 针对溢出部分
